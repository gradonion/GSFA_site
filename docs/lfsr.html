Local False Sign Rate Estimate of KO Effect on Genes

Yifan Zhou (zhouyf@uchicago.edu)

2020-11-12

For each factor analysis result, we estimated the local false sign rate (lfsr) of a certain knock-out effect on each gene based on the posteriors of \(\beta\) and \(W\).

For gene \(j\) and KO condition \(m\), \[lfsr_{mj} = \text{min} \Big\{\text{Pr}(\sum_{k=1}^K \beta_{mk}W_{jk} \geq 0 \text{ | Data}), \text{Pr}(\sum_{k=1}^K \beta_{mk}W_{jk} \leq 0 \text{ | Data}) \Big\}\] Under each KO condition, the number of genes that pass a certain lfsr cutoff would be compared to that from a simple Wilcoxon DGE analysis.
